# Avrajit Ghosh Academic Homepage

This is a plain HTML/CSS GitHub Pages homepage.

## Local Edits

- Edit `index.html` for text, publications, links, and news.
- Edit `style.css` for colors, spacing, fonts, and layout.
- Put your photo at `assets/profile.png`.
- Put your CV at `assets/Avrajit_Ghosh_CV.pdf`.

## Publish

Create a GitHub repository named `Avra98.github.io`, then push this folder:

```bash
cd Avra98.github.io
git init
git add .
git commit -m "Create academic homepage"
git branch -M main
git remote add origin https://github.com/Avra98/Avra98.github.io.git
git push -u origin main
```

The site will be served at:

```text
https://avra98.github.io/
```
